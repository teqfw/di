Simple Application

"s001" code

Constructors

Sample_App

All deps are instances (created once, stored in container): 
constructor({Sample_App_Config$, Sample_App_HandlerA$, Sample_App_HandlerB$})

Sample_App_Config

Has no deps: 
constructor()

Sample_App_HandlerA

Awaits to new `Sample_App_Service` instance: 
constructor({Sample_App_Service})

Sample_App_HandlerB

Awaits to new `Sample_App_Service` instance: 
constructor({Sample_App_Service})

Sample_App_Service

Awaits to instance of `Sample_App_Service` created before (for `Sample_App`): 
constructor({Sample_App_Config$})

Running

Open browser console to see console log. Expected output:

'Sample_App_Config' instance is created.
'Sample_App_Service' instance is created.
'Sample_App_HandlerA' instance is created.
'Sample_App_Service' instance is created.
'Sample_App_HandlerB' instance is created.
Application 'Sample_App' is running (deps: [Sample_App_Config, Sample_App_HandlerA, Sample_App_HandlerB]).
Service running with 'Sample_App_HandlerA' param.
Service running with 'Sample_App_HandlerB' param.
