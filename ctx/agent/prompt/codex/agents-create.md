# Создание AGENTS.md для отдельного уровня проекта

В когнитивном контексте появился новый уровень документов: `PATH`. Нужно создать файл `AGENTS.md` в этом каталоге.

## Дополнительные требования

- Предварительно проанализируй все файлы `AGENTS.md` вверх по иерархии до самого корня проекта, чтобы соответствовать требованиям, в них изложенным.
- Проверь документы, содержащиеся в указанном каталоге (только в нём!) на семантику - есть ли в них какие-то противоречия или ошибки для текущего уровня. Не надо проверять совместимость документов уровня со всем корпусом документов проекта. Совместимость нужна только внутри уровня.
